Reddit Data Pipeline

The project is an automated data pipeline designed to extract, process, and analyze large volumes of Reddit data for advanced analytics and dashboarding. With the growing importance of social media insights, organizations require scalable and modular solutions to efficiently collect, transform, and store unstructured data from external APIs. This project addresses these needs by streamlining the entire ETL process, ensuring data integrity, scalability, and ease of integration with analytics platforms.

This project utilizes Dockerized Apache Airflow for workflow orchestration and scheduling, and PostgreSQL as the staging relational database. Data is extracted from the Reddit API, staged in PostgreSQL, and then automatically ingested into Amazon S3. Data transformation and querying are performed using AWS Glue and Amazon Athena, with the final processed data loaded into Amazon Redshift to enable robust analytics and dashboarding capabilities.

Personal AI Mailer Assistant

This project is an intelligent, automated mailer that bridges the convenience of Telegram messaging with the power of AI-driven email composition and delivery. By simply sending a natural language instruction to a custom Telegram bot, users can have professional emails drafted by ChatGPT and sent directly from their Gmail account to any recipient. The workflow is seamless: the Telegram bot receives the user's message, extracts the recipient's email address and message intent, leverages OpenAI's GPT-4o model to generate a polished email draft, and then dispatches the email using the Gmail APIâ€”all in real time and without manual intervention.

The tech stack for this project is robust and modern. It utilizes the Telegram Bot API (via the python-telegram-bot library) for user interaction, OpenAI's GPT-4o for natural language processing and email drafting, and the Gmail API (with OAuth2 authentication) for secure email delivery. The backend is orchestrated in Python, leveraging asynchronous programming for efficient message handling and API communication. This integration of messaging, AI, and cloud-based email services creates a powerful tool for effortless, professional communication.
