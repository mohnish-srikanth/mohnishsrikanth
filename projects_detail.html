A brief description of my major projects

Virtual Assistant

The project is an intelligent personal assistant that uses concepts of NLP and Deep learning to recognize commands to complete the user's task. Many of the existing virtual assistants have chosen to use a speech interface over a visual one, making it disadvantageous for the deaf and dumb. This project overcomes this disadvantage by introducing visual assistance. This way, the user can either choose to provide commands by voice or sign language.

This project uses Google Speech Recognition model for speech recognition, MySQL for the database module, frontend was developed with flask. The gesture recognition module uses convolutional neural networks and was trained using an extensive Americal Sign Langauge dataset.

Stock Prediction Using Machine Learning Algorithms

I built a project that analyses and predicts stock prices using advanced Machine Learning algorithms like Linear Regression, Decision Tree, Random Forest, Polyfit, and Deep Learning using Recurrent Neural Networks using LSTM architecture with an accuracy of 96.7%.

I also drafted a research paper in IEEE format around the conclusions and theory behind the same project.
